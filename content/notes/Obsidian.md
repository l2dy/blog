## Core Configuration

### Editor

Disable "Readable line length" for code blocks with long lines.

### Files and Links

Link format: Shortest path (the default)
Attachement folder path: `Garden/images`

### Templates

Template folder location: `Garden/templates`
Daily notes new location: `Calendar/notes`
Daily notes template: `Garden/templates/Templates, Properties, Daily Note`

### Hotkeys

1. Assign `Cmd+D` to "Open today's daily note".
2. In Files & Links, use shortest path possible in vault for new links.
